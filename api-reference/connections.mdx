---
title: "Connections"
description: "List, create, update, and delete sync connections between sources and destinations."
---

A connection links a source to a destination and defines how and when data syncs. Each connection specifies stream selection, sync modes, scheduling, and namespace configuration.

## Endpoints

| Method | Path | Description |
|---|---|---|
| `GET` | `/connections` | List connections |
| `POST` | `/connections` | Create a connection |
| `GET` | `/connections/{connectionId}` | Get connection details |
| `PATCH` | `/connections/{connectionId}` | Update a connection |
| `DELETE` | `/connections/{connectionId}` | Delete a connection |

## List connections

**`GET /connections`**

<ParamField query="workspaceIds" type="string[]">
  Filter by workspace UUIDs. Empty list returns all allowed workspaces.
</ParamField>

<ParamField query="tagIds" type="string[]">
  Filter by tag UUIDs. Empty list returns all connections.
</ParamField>

<ParamField query="includeDeleted" type="boolean" default="false">
  Include soft-deleted connections.
</ParamField>

<ParamField query="limit" type="integer" default="20">
  Maximum results to return. Range: 1–100.
</ParamField>

<ParamField query="offset" type="integer" default="0">
  Offset for pagination.
</ParamField>

```bash
curl --request GET \
  --url 'https://api.airbyte.com/v1/connections?workspaceIds=744cc0ed-7f05-4949-9e60-2a814f90c035' \
  --header 'Authorization: Bearer <token>'
```

## Create a connection

**`POST /connections`**

<ParamField body="sourceId" type="string" required>
  UUID of the source.
</ParamField>

<ParamField body="destinationId" type="string" required>
  UUID of the destination.
</ParamField>

<ParamField body="name" type="string">
  Optional name for the connection.
</ParamField>

<ParamField body="schedule" type="object">
  Schedule configuration.
  <Expandable title="properties">
    <ParamField body="scheduleType" type="string" required>
      Either `manual` or `cron`.
    </ParamField>
    <ParamField body="cronExpression" type="string">
      Cron expression. Required when `scheduleType` is `cron`.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="configurations" type="object">
  Stream selection and sync mode configuration.
  <Expandable title="properties">
    <ParamField body="streams" type="object[]">
      List of stream configurations.
      <Expandable title="stream properties">
        <ParamField body="name" type="string" required>
          Name of the stream.
        </ParamField>
        <ParamField body="namespace" type="string">
          Namespace of the stream.
        </ParamField>
        <ParamField body="syncMode" type="string">
          Sync mode. One of: `full_refresh_overwrite`, `full_refresh_overwrite_deduped`, `full_refresh_append`, `full_refresh_update`, `full_refresh_soft_delete`, `incremental_append`, `incremental_deduped_history`, `incremental_update`, `incremental_soft_delete`.
        </ParamField>
        <ParamField body="cursorField" type="string[]">
          Path to the cursor field. Required for incremental sync modes unless the source defines a default.
        </ParamField>
        <ParamField body="primaryKey" type="string[][]">
          Paths to primary key fields. Required for deduplication sync modes.
        </ParamField>
        <ParamField body="selectedFields" type="object[]">
          Fields to include. If omitted, all fields are synced.
        </ParamField>
        <ParamField body="mappers" type="object[]">
          Stream mappers to apply before writing to the destination.
        </ParamField>
      </Expandable>
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="namespaceDefinition" type="string">
  How to determine the destination namespace. One of: `source`, `destination`, `custom_format`. Default: `destination`.
</ParamField>

<ParamField body="namespaceFormat" type="string">
  Used when `namespaceDefinition` is `custom_format`. Use `${SOURCE_NAMESPACE}` to mirror the source namespace.
</ParamField>

<ParamField body="prefix" type="string">
  Prefix prepended to each stream name in the destination. Default: empty string.
</ParamField>

<ParamField body="nonBreakingSchemaUpdatesBehavior" type="string">
  How Airbyte handles non-breaking schema changes. One of: `ignore`, `disable_connection`, `propagate_columns`, `propagate_fully`. Default: `ignore`.
</ParamField>

<ParamField body="status" type="string">
  Connection status. One of: `active`, `inactive`, `deprecated`, `locked`.
</ParamField>

<ParamField body="tags" type="object[]">
  Tags to associate with the connection.
</ParamField>

<CodeGroup>

```bash cURL
curl --request POST \
  --url https://api.airbyte.com/v1/connections \
  --header 'Authorization: Bearer <token>' \
  --header 'Content-Type: application/json' \
  --data '{
    "sourceId": "95e66a59-8045-4307-9678-63bc3c9b8c93",
    "destinationId": "e478de0d-a3a0-475c-b019-25f7dd29e281",
    "name": "Postgres-to-Bigquery",
    "schedule": {
      "scheduleType": "cron",
      "cronExpression": "0 0 * * *"
    }
  }'
```

</CodeGroup>

## Update a connection

**`PATCH /connections/{connectionId}`**

<ParamField path="connectionId" type="string" required>
  UUID of the connection.
</ParamField>

<ParamField body="name" type="string">
  New name for the connection.
</ParamField>

<ParamField body="schedule" type="object">
  Updated schedule configuration.
</ParamField>

<ParamField body="configurations" type="object">
  Updated stream configurations.
</ParamField>

<ParamField body="status" type="string">
  Updated connection status.
</ParamField>

<ParamField body="nonBreakingSchemaUpdatesBehavior" type="string">
  Updated non-breaking schema update behavior.
</ParamField>

```bash
curl --request PATCH \
  --url https://api.airbyte.com/v1/connections/9924bcd0-99be-453d-ba47-c2c9766f7da5 \
  --header 'Authorization: Bearer <token>' \
  --header 'Content-Type: application/json' \
  --data '{"status": "inactive"}'
```

```bash
  --url https://api.airbyte.com/v1/connections/9924bcd0-99be-453d-ba47-c2c9766f7da5 \

```
