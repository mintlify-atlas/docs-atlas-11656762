## What is Airbyte?

Airbyte is a data integration platform that solves the long tail of data movement. It enables data engineers to move data from any source (databases, APIs, files) to any destination (data warehouses, lakes, databases) with minimal setup.

## Platform Components

The Airbyte Platform is built from several core components working together:

| Component | Description |
|-----------|-------------|
| **airbyte-server** | The main Configuration API server handling all platform management |
| **airbyte-workers** | Executes sync jobs using Temporal workflow orchestration |
| **airbyte-webapp** | React/TypeScript web interface for managing connections |
| **airbyte-bootloader** | Runs database migrations and environment setup at startup |
| **airbyte-api** | OpenAPI-defined public REST API for programmatic access |
| **airbyte-featureflag** | Feature flag management with LaunchDarkly and config-file support |
